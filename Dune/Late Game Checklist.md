# Late Game Checklist

- Where are my next 1-3 victory points?
- Can anyone end the game this round?
- Which players can buy The Spice Must Flow?
- Which alliances can change hands?
- Which conflict rewards are worth all-in combat?
- Who has unrevealed combat potential?
- Who has enough intrigue cards to threaten surprise points?
- Am I spending resources that should be saved for a guaranteed point?
- In Uprising, can worms double a reward into a game-ending swing?
- In expansion games, are tech, research, contracts, or objectives creating hidden point routes?
