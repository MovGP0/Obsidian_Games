# Late Game Checklist

- Where are my next 1-3 victory points? See [[Dune/Game Phase Identification|Game Phase Identification]].
- Can anyone end the game this round?
- Which players can buy [[Dune/The Spice Must Flow|The Spice Must Flow]] through [[Dune/Persuasion|persuasion]]?
- Which [[Dune/Faction Tracks and Alliances|alliances]] can change hands?
- Which [[Dune/Combat and Conflicts|conflict]] rewards are worth all-in combat?
- Who has unrevealed combat potential?
- Who has enough [[Dune/Intrigue Cards|intrigue cards]] to threaten surprise points?
- Am I spending [[Dune/Resource Economy|resources]] that should be saved for a guaranteed point?
- In [[Dune/Uprising|Uprising]], can worms double a reward into a game-ending swing?
- In expansion games, are [[Dune/Rise of Ix|tech]], [[Dune/Immortality|research]], [[Dune/Bloodlines|contracts]], or objectives creating hidden point routes?
