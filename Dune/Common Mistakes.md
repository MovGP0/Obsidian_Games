# Common Mistakes

| Mistake | Why it hurts |
|---|---|
| Buying too many mediocre [[Dune/Deck Building and Card Quality|cards]] | Bloats the deck and delays useful draws |
| Ignoring [[Dune/Swordmaster|Swordmaster]] | Leaves you with fewer [[Dune/Agents and Worker Placement|actions]] for the rest of the game |
| Getting [[Dune/Swordmaster|Swordmaster]] automatically | Sometimes delays more direct points |
| Fighting every [[Dune/Combat and Conflicts|conflict]] | Burns troops before point conflicts matter |
| Spreading [[Dune/Faction Tracks and Alliances|faction influence]] randomly | Creates no alliance pressure and wastes actions |
| Hoarding [[Dune/Resource Economy|resources]] | Resources need conversion paths into points |
| Revealing too early | Gives up agent actions and board control |
| Revealing too late | Wastes [[Dune/Persuasion|persuasion]] or combat that mattered earlier |
| Ignoring [[Dune/Intrigue Cards|intrigue cards]] | Misses hidden point and combat swings |
| Treating [[Dune/Dune Imperium Base Game|base]], [[Dune/Rise of Ix|Ix]], [[Dune/Immortality|Immortality]], [[Dune/Uprising|Uprising]], and [[Dune/Bloodlines|Bloodlines]] as the same game | Each module changes the value of actions |
