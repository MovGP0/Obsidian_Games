# Common Mistakes

| Mistake | Why it hurts |
|---|---|
| Buying too many mediocre cards | Bloats the deck and delays useful draws |
| Ignoring Swordmaster | Leaves you with fewer actions for the rest of the game |
| Getting Swordmaster automatically | Sometimes delays more direct points |
| Fighting every conflict | Burns troops before point conflicts matter |
| Spreading faction influence randomly | Creates no alliance pressure and wastes actions |
| Hoarding resources | Resources need conversion paths into points |
| Revealing too early | Gives up agent actions and board control |
| Revealing too late | Wastes persuasion or combat that mattered earlier |
| Ignoring intrigue cards | Misses hidden point and combat swings |
| Treating base, Ix, Immortality, Uprising, and Bloodlines as the same game | Each module changes the value of actions |
