# Bloodlines

Bloodlines is an expansion compatible with both base Dune: Imperium and Dune: Imperium - Uprising.

## What It Adds

- New leaders.
- Sardaukar commanders.
- Contracts and contract-related content.
- New Imperium and Intrigue cards.
- Additional compatibility modules depending on whether you use base Dune: Imperium or Uprising.

## Strategic Shift

Bloodlines deepens existing lanes rather than replacing the whole game. It adds more leader variety and more module-specific incentives, especially around contracts and commander effects.

## Practical Notes

- Separate Bloodlines evaluation by base game: base Dune: Imperium and Uprising do not value all additions the same way.
- Contracts are strongest when they align with what your deck already wants to do.
- Do not add modules casually if your group is still learning the base rhythm.

## Bludgeon Transcript Notes

Bludgeon's Bloodlines reveal/review transcripts are more enthusiastic about Bloodlines as an Uprising companion than as an automatic upgrade for original Dune: Imperium.

Key strategic implications:

- Sardaukar commanders make combat planning more persistent across turns.
- More contracts mean contract completion can become a real scoring lane instead of a side bonus.
- New cards and tech should be evaluated by route fit: combat, trashing, intrigue, faction pressure, or contract support.
- For original Dune: Imperium, Bloodlines needs a compatibility check because not every Uprising-facing incentive has the same value on the base board.
