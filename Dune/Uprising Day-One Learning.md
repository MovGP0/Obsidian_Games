# Uprising Day-One Learning

Day-one Uprising learning is about avoiding base-game autopilot. [[Dune/Uprising|Uprising]] keeps the Dune: Imperium skeleton, but spies, contracts, worms, battle icons, and board changes alter the value of ordinary turns.

Bludgeon's Uprising primer and digital-announcement coverage both frame Uprising as a system update, not a small rules patch.

## First Things To Learn

- [[Dune/Uprising|Spies]] are access tools and timing tools.
- Contracts create small objectives that can shape the early economy.
- Worms can turn conflict rewards into game-ending swings.
- Battle icons and objectives add public long-term incentives.
- The new board changes familiar routes to [[Dune/Swordmaster|Swordmaster]], [[Dune/High Council|High Council]], and resource conversion.

## Day-One Mistakes

- Treating worms like ordinary combat strength.
- Forgetting that spies can infiltrate occupied spaces.
- Using one spy as if it can pay multiple effects.
- Ignoring contracts because they look like side quests.
- Assuming [[Dune/Rise of Ix|Ix]] or [[Dune/Immortality|Immortality]] modules retain the same value when mixed with Uprising.

## Learning Order

1. Learn spy placement, infiltrate, gather intelligence, and recall timing.
2. Learn contracts as early-game goals.
3. Learn worm requirements and Shield Wall restrictions.
4. Track battle icons and objectives.
5. Only then start adding cross-compatible expansion modules.
